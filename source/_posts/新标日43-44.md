## 陳さん、息子をアメリカに留学させます

### 文法

#### 使役形式

| 动词类别 | 基本形 | 使役形式   |
| -------- | ------ | ---------- |
| 一类动词 | 書く   | 書かせる   |
|          | 急ぐ   | 急がせる   |
|          | 読む   | 読ませる   |
|          | 話す   | 話させる   |
| 二类动词 | 食べる | 食べさせる |
|          | 見る   | 見させる   |
| 三类动词 | 来る   | こさせる   |
|          | する   | させる     |

#### 名詞 は 名詞 を　自动 させます

> 部長は 李さんを 出張させます。
>
> 部长让小李出差。

> 田中さんは 公園で 子供を 遊ばせています。
>
> 田中让小孩在公园玩。
