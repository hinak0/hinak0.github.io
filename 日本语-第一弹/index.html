其他发音规则

  • 促音 「っ」是用来表示停顿的符号，它比「つ」小一号，不发音，但是占一拍

  • 长音

声调

规律：

  • 第一个音和第二个调子不会相同
  • 不会出现高低高的音调
  • 第几个音降下去，就称为几型

examples:

  • 0 型 庭「にわ」\ 壁 \ 国「くに」 \ 椅子「いす」
  • 1 型 猫 \ 母 \ 傘 \ 外「そと」
  • 2 型 花屋「はなや」 \ います \ 寝ます「ねます」 \ 蕎麦屋「そばや」
  • 3 型 ください \ 食べます \ 聞きます「ききます」 \ 読みます「よみます」
